# ⚙️ Installation

Clone repository:

```bash
git clone https://github.com/VIDIT-13/portfolio.git
```

Move into directory:

```bash
cd portfolio
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```
