# ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/your-username/portfolio.git
```

Move into project folder:

```bash
cd portfolio
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```
